workspace\project\contact-form-7-dynamic-text-extension\includes\shortcodes.php

Summary: This plugin provides a comprehensive set of shortcodes (prefixed with 'CF7_') to easily retrieve dynamic data from the WordPress environment. These shortcodes allow developers to fetch various pieces of information, including GET/POST variables, the current URL, referrer URL, blog information, post variables (title, ID, custom fields), current user data, attachment details, cookie values, taxonomy terms, theme option values, and generate a unique GUID. The system utilizes extensive filtering for sanitization and escaping of the returned data.
