workspace\project\contact-form-7-dynamic-text-extension\includes\utilities.php

Summary: This plugin provides a dynamic text extension (DTX) for Contact Form 7. It introduces custom filtering layers for sanitization, escaping, and protocol handling for dynamic field values. It includes functions to detect value types (URL, email, text), obfuscate values, validate post/user access to specific post meta keys and user data keys, and generates the necessary HTML markup for various form field types (input, textarea, select, checkboxes, and groups). The system allows administrators to create an allow-list for custom post meta keys and user data keys, controlling access rights dynamically.

workspace\project\contact-form-7-dynamic-text-extension\includes\admin\settings.php

Summary: This plugin manages plugin settings for a Contact Form 7 Dynamic Text Extension, allowing administrators to configure permissions for post metadata and user data access. It provides a settings page where users can define allow lists for specific metadata keys and user data keys. The core feature is a form scanning tool that checks CF7 forms for the use of dynamic shortcodes (CF7_get_custom_field and CF7_get_current_user) and reports which specific keys are utilized. The scan results page allows administrators to review detected keys and manually add them to the corresponding allow lists. The system also features pagination for scanning multiple forms.
